Act 3 — Grill Gate

The Grill Gate — Trading System

🏛 Senior Architect

Latency, Data Model, Scale
  • Can WS gateway handle 10K concurrent traders?
  • What if Redis pub/sub drops a signal?
  • Is the positions table correct for concurrent updates?
  • How do you guarantee <50ms signal delivery?

🛡 Security Reviewer

Auth, Financial Data, Compliance
  • API key rotation for trading clients?
  • Position data encrypted at rest?
  • Audit log on every trade action?
  • Rate limiting on signal subscription?

🔥 Staff Engineer

Risk, Circuit Breakers, Failures
  • What if the ML model returns garbage?
  • Circuit breaker on max drawdown?
  • Rollback plan for bad signal deploy?
  • What happens during a flash crash?
